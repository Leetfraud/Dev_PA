# DevProfile Analyzer

> GitHub Portfolio Intelligence Platform - Transform your GitHub profile into meaningful analytics

A modern, professional web application that analyzes GitHub profiles and provides portfolio insights, repository analytics, developer comparisons, and AI-powered improvement suggestions.

## 🎯 Features

- **Portfolio Analytics Dashboard** - Visual overview of your GitHub profile with score and statistics
- **Repository Ranking System** - Analyze repositories by stars, forks, and activity level
- **Language Distribution** - Visualize your technology stack and skill distribution
- **Developer Comparison** - Compare profiles side-by-side with other developers
- **Smart Suggestions** - Get personalized recommendations to improve your portfolio
- **Contribution Heatmap** - Track your activity patterns over time
- **Search History** - Quick access to recently searched profiles
- **Professional UI** - Modern SaaS design with glassmorphism and smooth animations

## 🚀 Quick Start

### Prerequisites
- Node.js 16+ and npm 8+

### Installation

```bash
npm install
npm run dev
```

Visit `http://localhost:5173`

## 📊 Built With

- React 18 • Vite • Tailwind CSS
- Chart.js • Axios • Lucide React
- GitHub REST API
